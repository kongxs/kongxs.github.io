3.1 transform的概念

google 的android编译流程，以transform的形式提供给我们可干涉的时机
我们可以在编译成class文件并且生成dex之前拿到这些编译完成的class文件，做一些我们想要的修改。
我们自定义的transform会被包装秤一个 transform task，在 java compile task 之后执行。
